# 19/07/2026

## Hermes Obsidian skill — named vault support

Updated the Hermes Obsidian skill to support multiple vaults via `OBSIDIAN_VAULT_<NAME>` env vars:

- **Primary vault:** `OBSIDIAN_VAULT_PATH` — now set to `/home/hermes/homelab/vault/`
- **Homelab vault:** `OBSIDIAN_VAULT_HOMELAB=/home/hermes/homelab/vault/`

The skill now resolves named vaults when prefixed ("my homelab vault", "my uni vault", etc.) and falls back to the primary `OBSIDIAN_VAULT_PATH` for unqualified requests.
