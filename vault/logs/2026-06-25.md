# Hardware

- 16gb ram stick (2666) from ebay
- put in slot 3 and set speed to 2666 in bios
- pc started to boot loop so reset to auto (2133)
- slower but way more capacity

# Home Assistant

- shutdown vm
- hardware -> memory -> double click and increase from 2048 to 3072 (3 GiB)
- start vm

# Crafty Controller
