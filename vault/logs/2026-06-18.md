# Container Installs

## Crafty Controller

- ran proxmox helper script
- create folder in /mnt/storage in host
	- `mkdir -p /mnt/storage/crafty-backups`
- change permissions to make it accessible
```shell
chown -R 101000:101000 /mnt/storage/crafty-backups
chmod -R 775 /mnt/storage/crafty-backups
```
- add hard drive to lxc
	- `pct set 103 -mp0 /mnt/storage/crafty-backups,mp=/mnt/backups`
- go to ip WITH **https**
- username and password is in ~/crafty-controller.creds
- installed playit.gg on crafty controller lxc (running as a systemd service)
- kept crashing so changed arguments
	- from `java -Xms1000M -Xmx3000M -jar fabric.jar nogui`
	- to `java -Xms1000M -Xmx3000M -XX:+UseZGC -XX:+ZGenerational -jar fabric.jar nogui`

### Voxy

- lots of trial and error but got it working
- on server install chunky, sodium, voxy and voxy server
- on client install sodium, voxy and voxy server

- generate chunks with chunky (remove / if using server console)
	- `/chunky radius 1024` (or more)
	- `/chunky start`
- import existing chunks
	- `/voxyserver import existing all`
