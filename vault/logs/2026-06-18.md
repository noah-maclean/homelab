# Container Installs

## Crafty Controller

- ran proxmox helper script
- create folder in /mnt/storage in host
	- `mkdir -p /mnt/storage/crafty-backups`
- change permissions to make it accessible
```shell
chown -R 101000:101000 /mnt/storage/crafty-backups
chmod -R 775 /mnt/storage/crafty-backups
```
- add hard drive to lxc
	- `pct set 103 -mp0 /mnt/storage/crafty-backups,mp=/mnt/backups`
- go to ip WITH **https**
- username and password is in ~/crafty-controller.creds
- installed playit.gg on crafty controller lxc (running as a systemd service)
