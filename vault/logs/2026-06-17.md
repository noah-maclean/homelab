# Container Installs

## Immich

- advanced helper script install
	- 2 cores
	- 6 GB RAM
	- 20 GB HDD
	- GPU passthrough automatically detected
- change IP to 192.168.1.23 and change RAM to 4 GB
- make immich folder in Proxmox Host shell
	- `mkdir -p /mnt/storage/immich-photos/upload`
- mount storage folder in Proxmox Host shell
	- `pct set 102 -mp0 /mnt/storage/immich-photos,mp=/mnt/immich-photos`, where 102 is container id
- edit .env (`/opt/immich/.env`) in container
	- change `IMMICH_MEDIA_LOCATION` to `immich-photos` folder from previous step
- allow read and write access in Proxmox host
	- `chmod 777 /mnt/storage/immich-photos`
- copy initially created files:
	- `cp -R /opt/immich/upload/* /mnt/immich-photos/`
- change ownership in Proxmox host to allow unprivileged container to access HDD
	- `chown -R 101000:101000 /mnt/storage/immich-photos`
- update symlinks
```shell
rm /opt/immich/app/upload
rm /opt/immich/app/machine-learning/upload

ln -s /mnt/immich-photos /opt/immich/app/upload
ln -s /mnt/immich-photos /opt/immich/app/machine-learning/upload
```
- restart immich
	- `systemctl restart immich-web immich-ml`

### Problems

- failed installing one of the dependencies
	- disabled pihole to fix this
- segfaulted when installing immich
	- increased RAM and swap to install
	- decrease if install works
