# Container Installs

## Immich

- advanced helper script install
	- 2 cores
	- 4 GB RAM
	- 20 GB HDD
	- GPU passthrough automatically detected
- change IP to 192.168.1.23
- mount 1 TB HDD in LXC
- make immich folder
	- `mkdir -p /mnt/media-storage0/immich-photos`
- edit .env (`/opt/immich/.env)
	- change `IMMICH_MEDIA_LOCATION` to `immich-photos` folder from previous step
- copy initially created files:
	- `cp -R /opt/immich/upload/* /mnt/media-storage0/immich-photos/`
- fix permissions
	- `chown -R immich:immich /mnt/media-storage0/immich-photos`
- update symlinks
```shell
rm /opt/immich/app/upload
rm /opt/immich/app/machine-learning/upload

ln -s /mnt/media-storage1/immich-photos /opt/immich/app/upload
ln -s /mnt/media-storage1/immich-photos /opt/immich/app/machine-learning/upload
```
- restart immich
	- `systemctl restart immich-web immich-ml`
###  Result

- immich container has 20 GB from the SSD and folder on the 1 TB HDD used to store photos
